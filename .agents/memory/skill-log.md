# Skill Usage Log

> Track which skills are used and whether they helped. Update after each session.

## How to use

After completing a task that used a skill, add a row:

```
| DATE | SKILL | TASK | HELPED? | NOTES |
```

- `HELPED?`: `Y` = skill saved time, `N` = skill was not useful, `?` = unclear
- Delete rows older than 30 days

## Log

| Date       | Skill | Task                         | Helped | Notes                             |
| ---------- | ----- | ---------------------------- | ------ | --------------------------------- |
| 2026-06-08 | —     | Initial setup (this session) | N      | Skills were created, not consumed |

## Thresholds

If a skill gets 3+ `N` ratings in 30 days → delete it or rewrite.
If a skill gets 0 uses in 30 days → delete it.
If a pattern keeps being needed with no skill → create a new skill.
