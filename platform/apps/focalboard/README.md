# Focalboard - Project Management

## Purpose

Focalboard provides Kanban boards, tables, and calendars for project management integrated with the Fawkes platform.

## Key Features

- **Kanban Boards**: Visual workflow management
- **Table Views**: Spreadsheet-style data
- **Calendar Views**: Timeline planning
- **Custom Properties**: Flexible metadata
- **Templates**: Pre-configured board types

## Quick Start

### Accessing Focalboard

```bash
http://focalboard.127.0.0.1.nip.io
```

## Integration with Platform

- Link to GitHub issues
- Sync with Backstage catalog
- Webhook notifications to Mattermost
