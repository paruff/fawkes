üöÄ Getting Started

For Beginners (No platform experience)

Start with Module 1: Internal Delivery Platforms and progress sequentially through White Belt.

For DevOps Engineers (Some CI/CD experience)

Start with Module 5: Continuous Integration Fundamentals (Yellow Belt) or take the placement assessment.

For Platform Engineers (Experienced)

Jump to Module 17: Platform as a Product (Black Belt) or focus on specific skill gaps.

Lab Environment

Each module includes hands-on labs in isolated Kubernetes namespaces. Labs are auto-provisioned when you start a module and include:

  • Personal namespace with resource quotas
  • Pre-deployed sample applications
  • Automated validation and feedback
  • Real-time DORA metrics tracking

Assessment & Certification

Each belt requires:

  • Complete all modules in the belt
  • Pass hands-on practical exam
  • Pass written assessment (80-90% depending on belt)
  • Black Belt also requires: platform design project, peer review, code contribution, and mentoring 2 learners
