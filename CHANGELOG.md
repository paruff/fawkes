# Changelog

All notable changes to the Fawkes platform will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Comprehensive documentation structure with knowledge-base organization
- Implementation summaries directory for historical tracking
- Security Plane with SBOM, signing, and policy enforcement
- Dojo learning system with belt progression
- DORA metrics automation and dashboards
- Multi-cloud support (AWS, Azure, GCP)
- GitOps workflows with ArgoCD
- Observability stack (Prometheus, Grafana, OpenTelemetry)
- Jenkins CI/CD with golden path templates
- Backstage developer portal
- Mattermost collaboration platform
- Focalboard project management

### Changed

- Migrated documentation to structured knowledge base
- Updated README with clearer navigation and getting started
- Reorganized root-level markdown files

## [0.1.0-alpha] - 2025-01-26

### Added

- Initial platform foundation
- Jenkins Configuration as Code for Kubernetes pod/slave support
- Basic CI/CD pipeline structure
- Infrastructure as Code with Terraform
- Kubernetes orchestration setup
