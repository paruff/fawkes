# White Belt — What Is an Internal Developer Platform?

The White Belt is the starting point for the Fawkes Dojo learning journey. It covers
foundational concepts: what an Internal Developer Platform (IDP) is, why IDPs matter,
and how DORA metrics measure delivery performance.

## Module Overview

| Module | Topic | Duration |
|--------|-------|---------|
| Module 01 — What Is an IDP? | IDP concepts, golden paths, platform thinking | 2–3 hours |

## Learning Objectives

After completing the White Belt module, you will be able to:
- Explain what an Internal Developer Platform is and what problem it solves
- Describe the four DORA metrics and what they measure
- Identify where Fawkes fits in the IDP landscape
- Navigate the Backstage developer portal
- Understand the golden-path service template

## Prerequisites

No prior platform engineering experience is required. You should be comfortable with:
- Basic Git operations (clone, commit, push, branch)
- Basic command-line usage
- Familiarity with at least one programming language

## Assessment

After completing the module exercises, schedule an assessment session with a
Yellow Belt or higher. The assessment is a 30-minute practical exercise where you
will demonstrate the concepts from the module.
