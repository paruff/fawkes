Welcome to the Fawkes Dojo

The Fawkes Dojo is a comprehensive platform engineering curriculum designed to transform you from a beginner to a Platform Architect. Through 20 hands-on modules organized by belt levels, you'll learn everything from GitOps fundamentals to multi-cloud strategies.

Learning Path: Start with White Belt (Modules 1-4) and progress through Yellow, Green, Brown, and Black Belt levels. Each module includes theory, hands-on labs, real-world examples, and knowledge checks.

🚀 Getting Started

For Beginners (No platform experience)

Start with Module 1: Internal Delivery Platforms and progress sequentially through White Belt.

For DevOps Engineers (Some CI/CD experience)

Start with Module 5: Continuous Integration Fundamentals (Yellow Belt) or take the placement quiz.

For Platform Engineers (Experienced)

Jump to Module 17: Platform as a Product (Black Belt) or focus on specific skills gaps.

Lab Environment

Each module includes hands-on labs. Use the command fawkes lab start --module [number] to launch the lab environment.
