Welcome to the Fawkes Dojo

The Fawkes Dojo is not a traditional course or documentation site. It's an immersive learning environment where you master platform engineering by building and operating real platform infrastructure in a production-like environment.

Through 20 hands-on modules organized by belt levels, you'll progress from Platform Fundamentals to Platform Architecture, covering all 24 DORA capabilities. Each module combines theory, demonstration, hands-on labs, and immediate feedback.

🚀 Getting Started

For Beginners (No platform experience)

Start with Module 1: Internal Delivery Platforms and progress sequentially through White Belt.

For DevOps Engineers (Some CI/CD experience)

Start with Module 5: Continuous Integration Fundamentals (Yellow Belt) or take the placement assessment.

For Platform Engineers (Experienced)

Jump to Module 17: Platform as a Product (Black Belt) or focus on specific skill gaps.

Lab Environment

Each module includes hands-on labs in isolated Kubernetes namespaces. Labs are auto-provisioned when you start a module and include:

  • Personal namespace with resource quotas
  • Pre-deployed sample applications
  • Automated validation and feedback
  • Real-time DORA metrics tracking

Assessment & Certification

Each belt requires:

  • Complete all modules in the belt
  • Pass hands-on practical exam
  • Pass written assessment (80-90% depending on belt)
  • Black Belt also requires: platform design project, peer review, code contribution, and mentoring 2 learners

Support & Community

  • 💬 Mattermost: #dojo-learners, #dojo-white-belt, #dojo-yellow-belt, etc.
  • 📖 Documentation: docs.fawkes.io/dojo
  • 🎥 Office Hours: Every Friday 2-3pm UTC (live Q&A with mentors)
  • 🐛 Report Issues: GitHub Issues
  • 🎓 Platform Engineering University: Co-branded learning paths and certification alignment

Learning Tips

  • ⏰ Time Commitment: Each module takes ~60 minutes. Plan for 8 hours per belt.
  • 🔄 Spaced Repetition: Concepts are reinforced across multiple modules - this is intentional!
  • 🤝 Cohort Learning: Join or form a study group - learning together accelerates mastery.
  • 📝 Practice: Complete all labs, even if they seem easy. Muscle memory is key.
  • 💡 Ask Questions: Use Mattermost channels actively - there are no silly questions.
  • 🎯 Focus on Understanding: Don't just follow instructions - understand the "why" behind each step.
